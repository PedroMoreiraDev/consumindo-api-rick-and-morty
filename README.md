### consumindo-api-rick-and-morty

Consumo da API do Rick and Morty pelo Backend utilizando Java.

### Rotas consumidas:
    /character/{id}
    /location/{id}
    /episodes

### Tecnologias utilizadas
    Java
    Spring Boot
    Webflux
    Git
    Postman

### Link da API original: https://rickandmortyapi.com/
